Safety Road – A Comprehensive Vehicle Services Application

Client: University Group Project
Category: Vehicle Services App
Date: October 2024
Website: View Full Report (PDF)

Safety Road is a collaborative university project developed to provide an integrated digital solution for vehicle-related services in Saudi Arabia. The mobile app enables users to buy, sell, rent vehicles, and request maintenance or emergency roadside assistance — all from a single platform.

As part of the development team, I worked closely with one teammate on several core components including:

  • Functional and Data Requirements
  • Hardware & Software Environment (Section 3.1.1)
  • Use Case Diagrams for key features
  • Software Architecture Document (SAD)
  • User Interface Design for the Buy, Sell, and Rent modules

The application supports both Arabic and English, and is compatible with Android and iOS platforms. It features a user-friendly interface, AI-assisted search, secure payment systems, and interactive service scheduling — all designed to simplify the automotive experience in Saudi Arabia.
